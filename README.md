# Reproducible Scripts for Gas-rich UDGs in Hyperconical Modified Gravity

This repository is part of the broader HMG reproducibility hub:

- Parent repository: [robertmonjo/hmg](https://github.com/robertmonjo/hmg)

This repository contains the reproducible scripts used to generate the main numerical comparisons presented in the manuscript:

> Gas-rich ultra-diffuse galaxies: alleviating the MOND tension with hyperconical modified gravity

Live repository: <https://github.com/robertmonjo/hmg-udg-repro>

Archived release: <https://doi.org/10.5281/zenodo.19321452>

The repository includes:

- `fit_hmg_udg.py`
  - Pure Python implementation with no third-party dependencies.
- `fit_hmg_udg.R`
  - Base R implementation with no extra packages required.
- `2026_UDG_in_HMG_v3_original.r`
  - Original R workflow retained for reference.
- `table2_repro.csv`
  - Machine-readable version of the manuscript Table 2.
- `table2_repro.tex`
  - LaTeX tabular fragment for Table 2.
- `vcirc_beardplot_repro.svg`
  - Reproducible beardplot-style figure generated by the scripts.

## What is reproduced

Both the Python and R scripts reproduce:

- the HMG chi-square scan as a function of the environmental scale parameter `s`,
- the asymptotic HMG branch,
- the Newtonian and MOND outer-radius comparisons,
- the one-sided 95% lower limit on `s`,
- the combined-tension statistics quoted in the manuscript,
- a table analogous to Table 2,
- a beardplot-like figure analogous to Figure 1.

Typical outputs are:

- `chi2_hmg_asymptotic ~ 17.13`
- `s_95_one_sided_lower ~ 43.9`
- `chi2_newton ~ 9.3`
- `chi2_mond_mls ~ 615.66`
- `chi2_comb_newton ~ 4.41`
- `chi2_comb_hmg ~ 9.92`

## How to run

Python:

```bash
python fit_hmg_udg.py
```

R:

```bash
Rscript fit_hmg_udg.R
```

Both scripts write their output files into the same directory as the script.

## Notes

- The scanned environmental parameter is denoted by `s`, in agreement with the final manuscript notation.
- The asymptotic HMG branch corresponds to `epsilon_s^2 -> 1/6`.
- The one-sided 95% lower limit uses `Delta chi^2 = 2.71`.
- The HMG chi-square uses the published asymmetric observational velocity errors.
- The combined-tension statistics use the same asymmetric model intervals quoted in the final manuscript table.
